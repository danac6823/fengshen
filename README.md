# 心中有豐盛，豐盛自來 — 離線記帳 App（V4）

單檔離線記帳 App，整合**記帳 · 核對 · 分析 · 資產 · 投資 · 保險 · 旅遊**。
資料只存在「這台裝置的瀏覽器」(localStorage)，**不會上傳雲端**；可全離線使用。

---

## 功能總覽

> **V4 新增**：交易明細搜尋與「待核對 / 未分類」一鍵篩選、點商家可編輯整筆(含改卡片)、批次改類別、刪除可復原、記帳後可撤銷與常用商家快速帶入、設定收進右上齒輪、記住上次用的卡、核對自動依帳單期間比對並可一鍵接著分類、店家分類記憶可管理(分店共用)、訂閱/固定支出偵測與扣款前提醒、保單**美元計價**、國外交易費自動歸類、備份提醒移至齒輪紅點。


### 記帳
- 數字鍵盤快速記一筆，三大類別（家用 / 個人 / 狗貓）+ 子分類，常用子分類置頂。
- **記憶店家分類**：記過的商家會自動帶出上次的子分類（依商家名稱前 4 字學習，最新一次為準）。
- **公司 / 可報帳**標記，並可填「**可報帳金額**」做**部分可報帳**（例如刷卡 1,000 其中 600 報帳）。
- **掃描發票**：相機掃 QR / 條碼、或貼上發票字串自動帶入。
- 連續記帳天數、每日記帳提醒。

### 核對（可在設定開關）
- 上傳信用卡帳單：**貼上 / CSV / 拍照 OCR** 自動解析。
- 支援多家銀行格式，含「逐筆標籤式」帳單（消費日 / 入帳日 / 消費金額：TWD / 商家 / 卡號末4碼），自動用消費日、略過入帳日與卡號行。
- **智慧比對**：自動對上你記的交易；支援**多對一**（帳單 1 筆 = 你拆記的多筆，金額相加相符）。
- 選卡後顯示該卡帳期（結帳日 / 繳款日 / 目前帳期），跟卡片設定走。

### 分析
- 支出占比與分類明細、每月支出趨勢、預算、固定 vs 變動支出。
- **可報帳追蹤**：待收 / 已收、標已收、產生報帳單（可複製 / CSV / 列印），明細可收合。
- 交易明細**分頁**（每頁 30 筆）。
- **國外交易費**自動歸「家用 / 金融手續費」。

### 資產
- 淨資產、淨資產走勢、資產配置圓餅。
- 財務健康（緊急預備金月數、負債比率）、儲蓄率（已自動扣除報帳與旅遊代墊他人部分）。
- 資產 / 負債編輯（餘額更新留歷史）、信用卡繳款。
- **退休金**、**保單**入口卡（點開為獨立管理頁）。

### 投資
- 以「交易」為基礎的持股（平均成本）、概覽 XIRR 年化報酬。
- 交割帳戶（可用資金；入金 / 出金 / 股利明細可收合）。
- 股利統整、已實現損益（年度）、定期定額（多扣款日、記錄扣款）。

### 退休金（可在設定開關）
- 未退休：退休所需總額、可累積、缺口與達成率、建議再投入。
- 已退休：可撐年數、安全度（綠 / 黃 / 紅）、提領紀錄（連動扣帳戶）。
- 可連結既有投資 / 保單，避免與淨資產重複計算。

### 保單（獨立頁）
- 保障型（壽險 / 醫療 / 意外 / 重疾 / 癌症）看保額；儲蓄 / 投資 / 年金型有保單價值（計入淨資產）。
- 欄位：保單號碼、被保人、起始 / 生效日、繳費年期、下次繳費日、受益人、備註。
- **現值自動增值**：填現值 + 基準日 + 年增率（宣告利率），現值依複利每年自動長。
- **已繳費期滿**：不再計年保費、不再提醒繳費。
- **解約連動**：解約金存入指定帳戶、保單退出淨資產並留紀錄，可還原。
- 內附「保單說明」可收合。

### 旅遊分帳（可在設定開關）
- 多幣別、分帳結算（誰付 / 該攤 / 誰欠誰）、刷卡可帶入待核對交易。

### 提醒與其他
- 信用卡繳費提醒：**繳款截止日 + 應繳金額**（用已結帳那一期、跨月帳期計算）。
- 保費繳款提醒、預算超支提醒。
- 隱私遮罩（金額以 •••• 顯示）、JSON 匯出 / 匯入備份、資料自動存檔。
- 底部顯示版本，方便確認線上是否為最新版。

## 注意
- Service Worker 需要 https 才會啟用。第一次開啟需連線下載元件（React 等），之後快取可離線。
- 資料只在「這台裝置」。換手機或清除瀏覽器資料會不見；搬移 / 備份請用 App 內「匯出備份」，新裝置再「匯入備份」。

## 改版（更新 App）
1. 用新版整個資料夾，重新上傳 / 覆蓋到原站台（GitHub 重新 commit、或 Netlify 拖一次）。
2. **重點**：`sw.js` 第二行的 `CACHE` 版號要更新（本版為 `fengsheng-V4`），舊快取才會失效、自動換新版。
3. 已安裝在主畫面者：需完全關閉 App 再重開，才會抓到新版。

## 用 GitHub Pages（網址固定）
1. 進 repo →「Add file → Upload files」→ 把資料夾**裡面的檔案**（index.html、manifest.json、sw.js、三個 icon）拖進去 → Commit（index.html 要在根目錄）。
2. Settings → Pages → Deploy from a branch → `main` / `/ (root)` → Save。
3. 等 1～2 分鐘，網址為 `https://你的帳號.github.io/fengsheng/`。手機開該網址 →「加入主畫面」。
4. 改版：重新上傳 index.html + sw.js（版號已更新）→ Commit，幾分鐘後自動更新。開 `?V4` 可避快取。
